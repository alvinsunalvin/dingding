# dingding
免费钉钉打卡,仅用于学习玩耍,请于24小时后自行删除.

使用需安装一个钉钉修改版和一个控制端

-----克隆位置-----

1,启动钉钉,进行一次正确打卡

2,钉钉切后台,启动Env,点击'克隆位置'


-----下次使用-----

3,启动钉钉切到后台,启动Env点击'开始模拟位置'

4,切回钉钉打卡
